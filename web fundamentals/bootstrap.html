Navbar

.navbar

This class will make the navigation align horizontal

.navbar-expand-{size}

The navigation in your website will be more flexible. We dont want to show everything when the screen is too small. So it will show as a toggle button and hide the navigation and then unhide when click.

.navbar-{dark/light}

Don't get confused when you indicate dark it should show white. Maybe the word itself refer to the background.

.bg-*

There so many to choose from: primary, shows color blue preferred to combine with .navbar-dark. secondary, gray family color works on both .navbar-{light/dark}. The following works the same as secondary but differs in background color: success is green, info is light blue, warning is orange, danger is red. white, shows color? owww you're wrong it shows color pink now. Just kidding it's white of course and preferred to combine with .navbar-dark.light, same as white. dark, is for black and transparent is transparent.

.navbar-brand

Yeah it's a website name and add some link to redirect in home.

.navbar-toggler

This is responsible for transforming your navigation into a button when device-width below minimum. data-target is refer to what content will this button have when activate. I will be skiping the data-toggle, aria-label, aria-controls and aria-expanded as this was advance and not suitable for HTML & CSS lesson. .navbar-toggler-icon this is so important because it will create 3 bar icon in your toggle.Don't forget to create id to refer to the list

.collapse & .navbar-collapse

This is important because this will be responsible for hiding your navigation
